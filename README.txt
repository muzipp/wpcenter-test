WP Center Test is a WordPress plugin that creates a custom endpoint and manages AJAX-driven user data fetching and display.

== Description ==

This a test plugin for WPCenter that creates a custom endpoint and pulls data from the JSONPlaceholder API.

Key Features:
- Custom WordPress endpoint creation.
- Fetches and displays user data from an external API.
- Interactive frontend user data display in a table format.
- Efficient caching of user data to optimize performance.
- AJAX request handling for user details and cache clearing.

== Installation ==

1. Upload the `wpcenter-test` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the designated page to see the plugin in action.
